Urban Footprint

You can create a GeoJson file in QGIS or in ArcGis. The file should have at least one polygon describing the urban footprint. Layer name is the name that will be displayed on the policy lever drop down menu. The file should include at least two attributes: ID (with a unique identifier) and DENSITY (describing the population density in inhabitants per hectare for each polygon).
